The easiest way to get match WordPress with Thinkific!

== Description ==

This Thinkific Extension for Groundhogg makes connecting WordPress to Thinkific a snap! Choose from the following benchmarks and supercharge your funnels!

* Track Signs Ups, Enrollments & Orders
* Create Student Accounts & Enrollments
* Monitor Progress & Lession Completion

== Installation ==

We recommend viewing our documentation on WooNinja.io for full installation steps.

Installation is a simple but remember to read all the steps, including how to set up Thinkific itself to report to your site.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Groundhogg->Settings->Thinkific screen to configure the plugin
4. Enter your license key, click Save Changes, then click Activate.
