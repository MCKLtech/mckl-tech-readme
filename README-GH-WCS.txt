== Installation ==

We recommend viewing our documentation on WooNinja.io for full installation steps.

Installation is a breeze but remember to read all the steps, including how to set up Mailgun itself to report metrics to your site.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Groundhogg->Settings->Subscriptions screen to configure the plugin
4. Enter your license key, click Save Changes, then click Activate.
